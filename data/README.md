# Data: CDC Diabetes Health Indicators

This folder contains the CDC Health Indicators dataset used for diabetes classification.

### Dataset Overview
- **Source:** CDC BRFSS 2015 survey via Kaggle.
- **Structure:** 253,680 rows x 22 columns.
- **Target:** `Diabetes_binary` (0: No Diabetes, 1: Pre-diabetes or Diabetes).

### Key Statistics
- **Class Imbalance:** 86.1% Non-diabetic vs 13.9% Diabetic cases.
- **Key Predictors:** HighBP, HighChol, BMI, GenHlth, Age.

Note: For detailed exploratory data analysis (EDA), please refer to the main README in the root directory.
